## 官方新版本发布流程

* 修改代码中版本号及修改日期
* 修改 package.json 与 bower.json 中的版本号
* 修改 changelog.md
* `grunt test`
* `grunt release`
* 提交代码合并到 master
* Github 生成 release 包（for bower）
* 发布到 npm（`npm publish`，需 npm 协作者身份）
